# secure-password-manager
A simple password manager built with Python that allows users to save, view, and generate secure passwords using file handling and dictionaries.

#  Personal Password Manager

A simple command-line Password Manager built using Python.

This project allows users to:
- Save passwords for different websites.
- View saved passwords.
- Generate strong random passwords.
- Store passwords in a text file for future use.

##  Features

- Save website passwords
- View all saved passwords
- Generate secure random passwords
- Store data using file handling
- Beginner-friendly Python project

##  Technologies Used

- Python
- Dictionary
- File Handling
- Random Module
- String Module

##  How to Run

1. Clone the repository.

```
git clone https://github.com/your-username/personal-password-manager-python.git
```

2. Open the project folder.

3. Run the program.

```
python main.py
```
